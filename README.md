# N-Queen Optimization Using Genetic Algorithm and Differential Evolution

This project focuses on solving the N-Queen problem using evolutionary optimization techniques.

## Overview

A probabilistic distribution algorithm (PDA) based on genetic algorithms was designed and implemented to address limitations of standard approaches. This project demonstrates algorithm design, performance evaluation, and optimization techniques for combinatorial problems.

## Features
- Solution of the N-Queen problem using evolutionary algorithms
- Genetic Algorithm (GA) implementation
- Differential Evolution (DE) implementation
- Probabilistic Distribution Algorithm (PDA)
- Performance comparison between GA, DE, and PDA

## Implementation
- Designed and implemented a probabilistic distribution algorithm (PDA) based on genetic algorithms  
- Compared performance with Genetic Algorithm (GA) and Differential Evolution (DE) methods  
- Demonstrated improved convergence behavior and reduced failure cases compared to standard GA  
- Evaluated algorithm performance through iterative simulations and analysis  
- Implemented the solution in Python using population-based optimization techniques  
- The algorithms are tested on the N=8 case and evaluated based on convergence performance.

## Technologies
- Python
- Evolutionary algorithms
- Optimization techniques
